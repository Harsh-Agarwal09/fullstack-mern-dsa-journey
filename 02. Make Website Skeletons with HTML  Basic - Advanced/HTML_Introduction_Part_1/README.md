* Primary used for documentation (google docs)
* Alternatives: XML, Markdown, Jinja

HTML elements are hierarchical in a nature which means they can be nested with each other to create a tree like structure of the content on web page.

DOM (Document Object Model) -> Use to render content

<!DOCTYPE html>

<html> -> root element

<head> -> metadata about the information in the web page

<body> -> where all the content resides

Paragraph Tag: Used to create Paragraphs
Heading Tag: <h1>, <h2>, <h3>, <h4>, <h5>, <h6>
Comments: Used to insert node in your web page
    Ctrl + / : Shortcut to comment
Table Tag:
List Tag:
    Unordered List
    Ordered List
    Description List
Line Break <br>: Use between lines to add a break
Pre Tag <pre>: To display a text as it is written in the same manner. Usually used while rendering code
<hr>: Adds a horizontal line

HTML Inline and Block Elements

Inline: <a>, <strong>, <span> - Always displayed on the same line
Block: <p>, <div>, <h1> - Occupy full width of the web page

Formatting Tags:
    Physical Tags: <i>, <b>
    Semantic Tags: <article>, <nav>, <header>
